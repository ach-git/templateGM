# templateGM

templateGM is template gamemode for fivem with player system and sql connection.

### Put in your cfg file:
```lua
# SQL CONNECTION
set mysql_connection_string "server=localhost;database=basegm;userid=root;password="

# Gamemode
ensure baseGM
```
